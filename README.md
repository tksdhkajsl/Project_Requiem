## 2. 프로젝트 컨셉

### 2.1 프로젝트 목표

#### 핵심 가치
- 소울라이크 장르의 근간인 **긴장감 있는 전투 메커니즘**을  
  Unreal Engine 5 환경에서 고밀도로 구현

#### 기술적 목표
- 플레이어, 일반 몬스터, 다수의 보스 시스템 간의  
  **유기적인 상호작용**
- 복잡한 전투 구조 속에서도 **안정적인 시스템 통합**

---

### 2.2 프로젝트 컨셉

#### 2.2.1 세계관

**아트레이시아(Artraisia)**  
검과 마법이 공존하는 세계로,  
인류의 *7대 선*과 *7대 죄*를 상징하는 두 신의 대립 아래 놓여 있습니다.

---

##### 성신 아리엘 (Ariel, The Radiant Order)
- 순백의 날개와 눈부신 광채를 지닌 신
- 인류가 **7대 선**을 통해 신성에 도달하기를 희망
- 질서와 절제를 상징

##### 악신 이라 (Ira, The Primal Surge)
- **7대 죄**를 상징하는 죄악의 군주
- 억눌린 욕망의 폭발과 본능의 해방을 상징
- 인류가 본능에 충실할 때 진정한 자유를 얻는다고 주장

---

#### 사건의 발단

이라의 추종자인 **네크로맨서**들이  
아트레이시아 대륙 **기드리안 왕국의 백장 영지** 한복판에  
`대죄의 씨앗 (악의의 근원)`을 심습니다.

- 씨앗은 인간의 억눌린 욕망  
  (시기, 질투, 식탐 등)을 증폭
- 사람들은 서로를 공격하게 되고
- 사기(死氣)가 충만해지며 지역 전체가 황폐화
- 저주와 괴물이 들끓는 땅으로 변모

그 무렵,  
**방랑기사인 플레이어**가 이 지역에 발을 들이면서  
이야기가 시작됩니다.

## 3. 조작법

### 기본 조작

| 입력 | 기능 |
|---|---|
| W / A / S / D | 플레이어 이동 |
| 마우스 이동 | 시야 회전 |
| 마우스 좌클릭 | 공격 |
| 마우스 우클릭 | 전방 몬스터 락온 (Lock-On) |
| Space Bar | 회피 롤 (무적 판정) |
| Left Shift | 달리기 토글 |
| E | 상호작용 |
| K | 플레이어 스탯 창 |
